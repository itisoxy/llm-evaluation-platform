# llm-evaluation-platform

Built an LLM response evaluation platform focused on AI output quality assessment.

The project allows evaluators to:
• compare AI responses
• score helpfulness and accuracy
• identify hallucinations
• assess tone and safety
• export structured evaluation data

This helped me better understand real-world AI training and human feedback workflows used in modern LLM systems.


## Environment Variables

Create a `.env.local` file:

OPENAI_API_KEY=your_api_key

If no API key or quota is available, the app automatically falls back to mock AI responses.
